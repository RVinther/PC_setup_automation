# PC_setup_automation ![PowerShell](https://img.shields.io/badge/PowerShell-Script-blue)

Et PowerShell-script, der automatiserer klargøring af en ny Windows-PC.

Scriptet installerer standardsoftware, opretter mapper, kopierer supportfiler, tester internetforbindelse, håndterer winget automatisk og gemmer en log over hele processen.

## Formål med projektet

At anvende PowerShell til noget praktisk og realistisk inden for IT-support og drift.

## Funktioner

- Tjekker internetforbindelse før installation
- Tjekker om winget er installeret og installerer det automatisk hvis nødvendigt
- Installerer standardprogrammer via winget
- Opretter faste mapper (C:\Logs, C:\Support, C:\Temp)
- Kopierer filer fra projektets "Filer"-mappe til C:\Support
- Ændrer Windows Explorer-indstillinger (viser filendelser og skjulte filer)
- Logger hele processen til C:\Logs\Setup_log.txt
- Stopper ved fejl

## Installerede programmer

Scriptet installerer som udgangspunkt: 

- Google Chrome
- 7-Zip
- VLC Media Player
- Dropbox
- Adobe Acrobat Reader (64-bit)
- Visual Studio Code
- Notepad++

Men listen kan nemt udvides eller tilpasses.
